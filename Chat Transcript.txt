# 1. Configure SQLite

Open:

```text
box_selector/settings.py
```

Find the `DATABASES` section.

For SQLite, it should look like this:

```python
DATABASES = {
 "default": {
 "ENGINE": "django.db.backends.sqlite3",
 "NAME": BASE_DIR / "db.sqlite3",
 }
}
```

### Explanation

- `ENGINE`
 - Tells Django which database to use.
 - Here it uses SQLite.

- `NAME`
 - Specifies the database file.
 - Django will automatically create `db.sqlite3` in your project folder when you run migrations.

Project structure after migration:

```text
box_selector/
│
├── db.sqlite3
├── manage.py
├── box_selector/
└── shipping/
```

No additional installation is needed because SQLite is built into Python.

# OrderItem Model

Stores products inside an order.

Instead of using a simple ManyToMany field, use an intermediate model because each product has a quantity.

## Fields

| Field | Type |
|--------|------|
| order | ForeignKey |
| product | ForeignKey |
| quantity | PositiveIntegerField |

---

### Django Model

```python
class OrderItem(models.Model):

 order = models.ForeignKey(
 Order,
 on_delete=models.CASCADE,
 related_name="items"
 )

 product = models.ForeignKey(
 Product,
 on_delete=models.CASCADE
 )

 quantity = models.PositiveIntegerField(
 validators=[MinValueValidator(1)]
 )

 class Meta:
 unique_together = ("order", "product")

 def __str__(self):
 return f"{self.product.name} x {self.quantity}"
```

`unique_together` ensures the same product cannot appear multiple times in the same order.

# Model Validations

Besides field validators, add model-level validation using the `clean()` method.

## Product Validation

```python
from django.core.exceptions import ValidationError

def clean(self):
 if self.length <= 0:
 raise ValidationError("Length must be greater than zero.")

 if self.width <= 0:
 raise ValidationError("Width must be greater than zero.")

 if self.height <= 0:
 raise ValidationError("Height must be greater than zero.")

 if self.weight <= 0:
 raise ValidationError("Weight must be greater than zero.")
```

---

## Box Validation

```python
def clean(self):

 if self.internal_length <= 0:
 raise ValidationError("Length must be greater than zero.")

 if self.internal_width <= 0:
 raise ValidationError("Width must be greater than zero.")

 if self.internal_height <= 0:
 raise ValidationError("Height must be greater than zero.")

 if self.max_weight <= 0:
 raise ValidationError("Maximum weight must be greater than zero.")

 if self.cost < 0:
 raise ValidationError("Cost cannot be negative.")
```

---

## OrderItem Validation

```python
def clean(self):

 if self.quantity <= 0:
 raise ValidationError("Quantity must be greater than zero.")
```

---

## Automatically Run Validation on Save

Django does **not** call `clean()` automatically when you call `save()`. To enforce validation consistently, override `save()` in each model:

```python
def save(self, *args, **kwargs):
 self.full_clean()
 super().save(*args, **kwargs)
```

  # `shipping/services/recommendation.py`
  
  ```python
  from decimal import Decimal
  
  from ..models import Product, Box
  
  
  def recommend_box(items):
   """
   Returns the best box according to the assignment rules.
  
   Rules
  
   1. Every product must fit inside the box.
   2. Total weight <= box capacity.
   3. Lowest cost wins.
   4. If cost is same -> Smaller volume wins.
   5. If still same -> Alphabetical name.
   """
  
   total_weight = Decimal("0")
  
   products = []
  
   # ------------------------------------
   # Fetch Products
   # ------------------------------------
  
   for item in items:
  
   product_id = item.get("product")
   quantity = item.get("quantity", 1)
  
   if quantity <= 0:
   return {
   "success": False,
   "message": "Quantity must be greater than zero."
   }
  
   try:
   product = Product.objects.get(pk=product_id)
  
   except Product.DoesNotExist:
  
   return {
   "success": False,
   "message": f"Product {product_id} does not exist."
   }
  
   total_weight += product.weight * quantity
  
   products.append(
   {
   "product": product,
   "quantity": quantity
   }
   )
  
   # ------------------------------------
   # Find Suitable Boxes
   # ------------------------------------
  
   suitable_boxes = []
  
   for box in Box.objects.all():
  
   # Weight check
  
   if total_weight > box.max_weight:
   continue
  
   fits = True
  
   # Check every product
  
   for item in products:
  
   product = item["product"]
  
   # Sort dimensions to allow rotation
   product_dimensions = sorted([
   product.length,
   product.width,
   product.height,
   ])
  
   box_dimensions = sorted([
   box.internal_length,
   box.internal_width,
   box.internal_height,
   ])
  
   if (
   product_dimensions[0] > box_dimensions[0]
   or product_dimensions[1] > box_dimensions[1]
   or product_dimensions[2] > box_dimensions[2]
   ):
   fits = False
   break
  
   if fits:
  
   volume = (
   box.internal_length
   * box.internal_width
   * box.internal_height
   )
  
   suitable_boxes.append(
   {
   "box": box,
   "volume": volume,
   }
   )
  
   # ------------------------------------
   # No Box Found
   # ------------------------------------
  
   if not suitable_boxes:
  
   return {
   "success": False,
   "message": "No suitable box found."
   }
  
   # ------------------------------------
   # Sort According to Assignment Rules
   # ------------------------------------
  
   suitable_boxes.sort(
   key=lambda item: (
   item["box"].cost,
   item["volume"],
   item["box"].name.lower(),
   )
   )
  
   best = suitable_boxes[0]
  
   return {
   "success": True,
   "recommended_box": best["box"].name,
   "box_id": best["box"].id,
   "cost": best["box"].cost,
   "volume": best["volume"],
   "max_weight": best["box"].max_weight,
   "total_weight": total_weight,
   "message": "Box recommended successfully."
   }
  ```

  # services/recommendation.py
  
  ```python
  from decimal import Decimal
  
  from django.db.models import F
  
  from shipping.models import Product, Box
  
  
  class BoxRecommendationService:
  
   def __init__(self, items):
   self.items = items
   self.products = []
   self.total_weight = Decimal("0.00")
  
   def recommend(self):
   """
   Main entry point.
   """
  
   self._load_products()
  
   suitable_boxes = self._find_suitable_boxes()
  
   if not suitable_boxes:
   return {
   "success": False,
   "message": "No suitable box found."
   }
  
   best_box = self._select_best_box(suitable_boxes)
  
   return self._build_response(best_box)
  
   # ---------------------------------------------------
  
   def _load_products(self):
  
   for item in self.items:
  
   product = Product.objects.get(pk=item["product"])
  
   quantity = item["quantity"]
  
   self.products.append(
   {
   "product": product,
   "quantity": quantity,
   }
   )
  
   self.total_weight += product.weight * quantity
  
   # ---------------------------------------------------
  
   def _find_suitable_boxes(self):
  
   suitable_boxes = []
  
   boxes = Box.objects.all()
  
   for box in boxes:
  
   if self.total_weight > box.max_weight:
   continue
  
   if self._products_fit(box):
  
   suitable_boxes.append(box)
  
   return suitable_boxes
  
   # ---------------------------------------------------
  
   def _products_fit(self, box):
  
   box_dimensions = sorted([
   box.internal_length,
   box.internal_width,
   box.internal_height,
   ])
  
   for item in self.products:
  
   product = item["product"]
  
   product_dimensions = sorted([
   product.length,
   product.width,
   product.height,
   ])
  
   if (
   product_dimensions[0] > box_dimensions[0]
   or product_dimensions[1] > box_dimensions[1]
   or product_dimensions[2] > box_dimensions[2]
   ):
   return False
  
   return True
  
   # ---------------------------------------------------
  
   def _select_best_box(self, boxes):
  
   return sorted(
   boxes,
   key=lambda box: (
   box.cost,
   box.internal_length
   * box.internal_width
   * box.internal_height,
   box.name.lower(),
   ),
   )[0]
  
   # ---------------------------------------------------
  
   def _build_response(self, box):
  
   volume = (
   box.internal_length
   * box.internal_width
   * box.internal_height
   )
  
   return {
   "success": True,
   "recommended_box": {
   "id": box.id,
   "name": box.name,
   "cost": box.cost,
   "volume": volume,
   "max_weight": box.max_weight,
   },
   "total_weight": self.total_weight,
   "message": "Box recommended successfully."
   }
  ```

  # Why this is better than the previous version
  
  ### 1. Single Responsibility Principle
  
  Instead of one long function, each method has a single responsibility:
  
  - `_load_products()` → Load products and calculate weight.
  - `_products_fit()` → Check whether all products fit inside a box.
  - `_find_suitable_boxes()` → Filter valid boxes.
  - `_select_best_box()` → Apply the assignment's sorting rules.
  - `_build_response()` → Format the API response.
  
  ---
  
  ### 2. Easy to Test
  
  You can write focused unit tests for each method, for example:
  
  ```python
  service._products_fit(box)
  service._select_best_box(boxes)
  service._load_products()
  ```
  
  rather than testing one large function.
  
  ---
  
  ### 3. Thin View
  
  The API view is responsible only for:
  
  - Reading the request.
  - Validating required input.
  - Calling the service.
  - Returning the HTTP response.
  
  All business logic stays in the service layer.
  
  ---
  
  ### 4. Reusable Service
  
  The same `BoxRecommendationService` can be reused from:
  
  - A REST API
  - Django Admin actions
  - Management commands
  - Celery tasks
  - Future GraphQL endpoints
  
  without duplicating logic.

  ## One improvement I'd recommend
  
  Your current algorithm checks **each product independently** against the box dimensions and verifies the **total weight**, but it does **not** check whether the combined products can actually be packed together in the box. The assignment wording can be interpreted either way, but in real-world packing this matters.
  
  If you want to make your solution stand out, you can add a simple packing heuristic, such as:
  
  - Sum the volumes of all products and ensure the total volume does not exceed the box volume.
  - Keep the existing per-product dimension checks.
  - Continue applying the selection rules (lowest cost → smallest volume → alphabetical name).
  
  This isn't a full 3D bin-packing algorithm, but it's a stronger approximation and demonstrates additional thought beyond the minimum requirements while remaining practical for an interview assignment.
