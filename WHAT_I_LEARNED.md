# What I Learned from This Assignment

## I learned the following during this assignment:

1. I learned how to configure a Django project through the settings.py file by managing installed apps, database settings, middleware, templates, and other project configurations.

2. I learned how project-level urls.py is used to register application URLs using the include() function.

3. I learned how to create database tables using Django models without writing SQL queries manually. I used the following commands to generate and apply database migrations:

   - python manage.py makemigrations
   - python manage.py migrate

4. I learned how to use the Django Admin Panel to manage application data. I also learned how to create a superuser using:

   - python manage.py createsuperuser

5. I learned how views.py contains the backend business logic and how it processes requests and returns responses for the APIs.

6. I learned how application-level urls.py is used to map API endpoints to views, making it easier to connect the backend with the frontend and test APIs using Postman.

7. I learned how to use Django REST Framework to build REST APIs. I also learned how serializers convert Django model objects into JSON responses and validate incoming request data.

8. Using these Django and Django REST Framework concepts, I successfully built the AI-Assisted Box Selection System.

9. I learned how to write and run unit tests for Django applications using:

   - python manage.py test

10. I learned the basics of Git and GitHub for version control and collaboration. I used commands such as:

    - git init
    - git add
    - git commit
    - git push

11. I learned the importance of writing proper project documentation, including API documentation, README files, AI usage documentation, and test output documentation.

12. While working on this assignment, I learned how to use AI tools more effectively by writing clear and specific prompts. I used ChatGPT as a development assistant to understand concepts, improve parts of the implementation, and resolve issues. I verified the code manually through testing and debugging before finalizing the project.
